## Setup:

1. Starte den Server via tomcat
`ch.fhnw.kvan.chat.servlet.server.Server`

2. Starte einzelne Clients
	1. `ch.fhnw.kvan.chat.general.RunClient`
	2. Wähle einen Client aus und drücke **Start**
